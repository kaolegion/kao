# KAO ↔ RAY — Cognitive Architecture Model

## Purpose

This document defines the long-term systemic relationship between:

- KAO — cognitive operating system
- RAY — operational agent layer

It is a structural doctrine document.

---

## Core principle

KAO is the cognitive environment.
RAY is an operational entity acting within this environment.

KAO must be able to exist:

- without RAY
- without external AI
- without network

RAY cannot exist cognitively without a host system.
