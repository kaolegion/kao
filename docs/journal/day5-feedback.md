# DAY 5 — FEEDBACK ORGAN

Kao starts to learn from outcomes.

Capabilities introduced:
- feedback recording on action results
- classification success / failure / neutral
- persistent feedback log
- feedback state summary

Command surface:
- kao feedback record <action_id> <result>
- kao feedback log
- kao feedback state

State path:
state/feedback/feedback.log

Outcome:
Kao can now associate actions with consequences.
